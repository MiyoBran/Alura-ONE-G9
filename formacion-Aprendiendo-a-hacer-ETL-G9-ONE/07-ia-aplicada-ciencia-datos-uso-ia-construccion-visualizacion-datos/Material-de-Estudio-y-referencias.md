# 📚 Materiales de Estudio y Referencias

## Introducción

El análisis y visualización de datos es un campo vasto que abarca múltiples disciplinas: educación, marketing, ventas y más. En esta formación exploramos el análisis de datos de ventas utilizando **Python** y **ChatGPT** como herramientas fundamentales.

A continuación encontrarás un listado de recursos curados que te permitirán profundizar en los temas cubiertos.

---

## 🐍 Librerías de Python para Análisis de Datos

### [Pandas](https://pandas.pydata.org/)
> **Tipo**: Biblioteca | **Costo**: Gratuito | **Idioma**: Inglés | **Formato**: Código Abierto

Biblioteca fundamental para manipulación y análisis de datos en Python. Proporciona estructuras de datos eficientes (DataFrames y Series) y herramientas de alto rendimiento para:
- Importar y exportar datos
- Limpiar y transformar datos
- Realizar análisis exploratorio
- Manipulación de datos tabulares

**Caso de uso**: Limpiar datos de ventas, calcular métricas, agrupar información.

---

## 📊 Librerías de Python para Visualización

### [Matplotlib](https://matplotlib.org/stable/)
> **Tipo**: Biblioteca | **Costo**: Gratuito | **Idioma**: Inglés | **Formato**: Código Abierto

La librería estándar para crear visualizaciones en Python. Permite generar:
- Gráficos estáticos, animados e interactivos
- Gráficos de líneas, barras, dispersión, histogramas
- Personalización completa de estilos, colores y anotaciones
- Integración perfecta con NumPy y Pandas

**Caso de uso**: Crear dashboards de análisis de ventas, tendencias de mercado.

---

### [Seaborn](https://seaborn.pydata.org/)
> **Tipo**: Biblioteca | **Costo**: Gratuito | **Idioma**: Inglés | **Formato**: Código Abierto

Librería basada en Matplotlib que simplifica la creación de visualizaciones estadísticas. Ofrece:
- Interfaz de alto nivel y código más limpio
- Gráficos estadísticos profesionales por defecto
- Mejor manejo de DataFrames de Pandas
- Temas visuales predefinidos

**Caso de uso**: Análisis exploratorio rápido, gráficos estadísticos complejos.

## 🎨 Recursos de Diseño y Colores

### [Nombres de Colores en Matplotlib](https://matplotlib.org/stable/gallery/color/named_colors.html)
> **Tipo**: Referencia | **Costo**: Gratuito | **Idioma**: Inglés | **Formato**: Web

Guía completa de colores predefinidos en Matplotlib incluyendo:
- Nombres de colores CSS

- Paletas de colores temáticas
- Ejemplos visuales

**Caso de uso**: Seleccionar colores coherentes para dashboards profesionales.

## 🤖 Herramientas con IA para Presentaciones

### [Gamma](https://gamma.app/)
> **Tipo**: Aplicación Web | **Costo**: Pago (con versión gratuita) | **Idioma**: Inglés | **Formato**: SaaS

Plataforma basada en IA para crear presentaciones, informes y documentos de forma rápida:
- Genera contenido automático a partir de prompts
- Interfaz intuitiva sin necesidad de diseño
- Personalización avanzada de temas y estilos
- Exportación en múltiples formatos

**Caso de uso**: Crear reportes ejecutivos y presentaciones profesionales de análisis.

## 📖 Cómo Aprovechar Estos Recursos

### Para Principiantes
1. Comienza con **Pandas** para aprender manipulación de datos
2. Usa **Matplotlib** para crear visualizaciones básicas
3. Consulta la guía de colores para mejorar estética

### Para Nivel Intermedio
1. Aprende **Seaborn** para visualizaciones más sofisticadas
2. Desarrolla dashboards con Matplotlib + Pandas
3. Experimenta con **Gamma** para reportes automáticos

### Para Nivel Avanzado
1. Combina todas las herramientas
2. Crea pipelines completos de análisis
3. Automatiza generación de reportes con IA
