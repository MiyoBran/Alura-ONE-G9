# 📈 Data Science: Probando Relaciones con Regresión Lineal

## 📖 Descripción

Curso fundamental en modelado predictivo usando regresión lineal. Aprenderás a identificar relaciones entre variables, construir modelos lineales y evaluar su calidad. Este curso es esencial para cualquier científico de datos.

Este curso es parte de la formación en **Estadística y Machine Learning** del programa **Oracle Next Education (ONE)** en colaboración con **Alura LATAM**.

---

## 🎯 ¿Qué vas a aprender?

✅ **Identificación de Relaciones:**
- Detectar relaciones lineales
- Variables explicativas vs respuesta
- Análisis de correlación
- Visualización de relaciones

✅ **Construcción del Modelo:**
- Ajustar regresión lineal
- Ecuación del modelo
- Interpretación de coeficientes
- Predicciones puntuales

✅ **Métricas de Calidad:**
- Coeficiente R²
- Error de predicción
- Análisis de residuos
- Validación del modelo

✅ **Factores Influyentes:**
- Multicolinealidad (VIF)
- Outliers
- Homocedasticidad y heterocedasticidad
- Mejora del modelo

---

- **Python 3.x**
