Estadisticas y Machine Learning G9 - ONE
Desbloquea el poder de la estadística y del aprendizaje automático: una inmersión profunda en el análisis de datos, la construcción de modelos y la toma de decisiones basada en datos.

¿Qué vas a aprender?
En esta formación aprenderás los conceptos esenciales de estadística, fundamentales para el campo de la Ciencia de Datos. Luego pasará a la regresión lineal, que le permitirá comprender cómo predecir eficazmente tendencias y modelar relaciones entre variables. Finalmente, explorará los fundamentos del aprendizaje automático, aprenderá cómo construir modelos predictivos y aplicar técnicas modernas de aprendizaje automático para resolver problemas del mundo real.

Esta formación forma parte del Programa ONE, una alianza entre Alura Latam y Oracle.

## Contenido del Programa

### 1. Plan de estudio
El plan de estudio es el primer paso para ayudarte a organizar tu rutina de estudio. En este plan de estudios encontrarás el orden sugerido en el que debe hacer los cursos y el tiempo estimado en el que debe completar cada módulo. Con él podrás seguir el ritmo del programa ONE, aprovechar al máximo tus tiempos de estudio y cada ruta de aprendizaje. El plan de estudio se realizó utilizando la herramienta Trello, la cual te permite gestionar cualquier tipo de proyecto y tareas, para utilizarla debes acceder al enlace, registrarte de forma gratuita y seguir las recomendaciones dejadas en el tablero de Trello para utilizar este plan de estudio.

Para más información sobre el plan de estudio, consulta el [tablero de Trello](https://trello.com/b/0WN5oLyo/formacion-estatistica-y-machine-learning).

### 2. Estadística
En esta etapa de Estadística, se presentarán los conceptos fundamentales esenciales para el análisis de datos. Aprenderás sobre estadísticas descriptivas, como medidas de tendencia central (media, mediana y moda) y dispersión (varianza y desviación estándar).

**Cursos:**
- Estadística con Python: frecuencias y medidas
- Estadística con Python: Probabilidad y muestreo
### 3. Regresión lineal
En esta etapa aprenderás a modelar datos utilizando regresión lineal. Comenzarás visualizando la distribución de datos para comprender patrones, distinguiendo entre variables dependientes y explicativas. Aplicarás regresiones utilizando herramientas como Statsmodels y Scikit-learn, interpretando coeficientes y utilizando métricas para evaluar la calidad del modelo.

**Cursos:**
- Data Science: probando relaciones con regresión lineal
- Regresión lineal: Técnicas avanzadas de Modelado
- Clasificación: aprendiendo a clasificar datos con Machine Learning
### 4. Machine learning: Comprender la base de la clasificación
Aprenderás cómo resolver problemas cotidianos mediante el aprendizaje automático y cómo aplicar Machine Learning para mejorar procesos empresariales. Realizarás un proyecto práctico para predecir retrasos en vuelos, poniendo en práctica todos los conocimientos adquiridos.

**Cursos:**
- Clasificación: validación de modelos y métricas de evaluación
- IA aumentada: previsión de atrasos de vuelos
