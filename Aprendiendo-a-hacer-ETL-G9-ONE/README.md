
# Aprendiendo a hacer ETL G9 - ONE

Domina el arte del ETL con Pandas: una guia completa para extraer, transformar y cargar datos con la libreria mas popular de Python. Esta formacion es parte de la escuela Data Science dentro del Programa ONE (Alura Latam + Oracle). Duracion estimada: 71h.

## Trayecto formativo (desde el punto 3)

### 3. Paquete de datos
Aprende a usar NumPy para manipulacion de matrices y operaciones numericas esenciales: calculos matematicos, estadistica basica, clasificacion, seleccion y mas.

**Recursos**
- NumPy: analisis numerico eficiente con Python

**Recursos**
- Pandas E/S: trabajando con diferentes formatos de archivo
- Pandas: transformacion y manipulacion de datos
- Pandas: seleccionar y agrupar datos
