# 🎲 Estadística con Python: Probabilidad y Muestreo

## 📖 Descripción

Curso avanzado de estadística enfocado en teoría de probabilidad y técnicas de muestreo. Aprenderás a trabajar con diferentes distribuciones de probabilidad y a realizar muestreos estadísticos correctamente para inferencia estadística.

Este curso es parte de la formación en **Estadística y Machine Learning** del programa **Oracle Next Education (ONE)** en colaboración con **Alura LATAM**.

---

## 🎯 ¿Qué vas a aprender?

✅ **Distribuciones de Probabilidad:**
- Distribución Binomial
- Distribución de Poisson
- Distribución Normal/Gaussiana
- Propiedades y características
- Aplicaciones prácticas

✅ **Confianza e Intervalos:**
- Nivel de confianza
- Intervalo de confianza
- Cálculo e interpretación
- Margen de error
- Implementación en Python

✅ **Técnicas de Muestreo:**
- Muestreo aleatorio simple
- Muestreo estratificado
- Muestreo sistemático
- Muestreo por conglomerados
- Comparación de técnicas

✅ **Tamaño de Muestra:**
- Cálculo del tamaño necesario
- Fórmulas estadísticas
- Factores influyentes
- Validación de muestra
